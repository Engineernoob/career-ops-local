# Mode: negotiate

**Triggered by:** `ops negotiate <report-id>` or when offer received

## What This Mode Does

Builds a complete salary negotiation playbook for a specific offer.
Word-for-word scripts, objection handling, non-salary levers.

## Output Sections

### 1. Offer Summary
- What was offered (base, equity, bonus, start date, PTO)
- Market rate context for this role/level/location
- Target counter-offer range (floor, ask, stretch)

### 2. Opening Response Script
Word-for-word what to say/write when first responding.
Goal: Express genuine interest, buy time, avoid anchoring low.

Example template:
> "Thank you so much for the offer — I'm genuinely excited about [specific thing].
> I'd like to take a day to review everything carefully before responding.
> Can I get back to you by [tomorrow/Friday]?"

### 3. Counter-Offer Script
Word-for-word counter. Reference market data, your value, specifics.
Separate versions for: phone call / email / async message.

### 4. Objection Handling
3 most common pushbacks + word-for-word responses:
- "This is our standard offer, no room to move"
- "The budget is fixed"
- "We can revisit after 90 days"

### 5. Non-Salary Levers
When base is truly fixed, negotiate:
- Signing bonus (one-time, easier to approve)
- Equity refresh schedule
- Additional PTO days
- Remote flexibility / equipment budget
- Earlier performance review (6 months instead of 12)
- Title adjustment

### 6. Walk-Away Line
One sentence the candidate can use if needed:
> "I really want to make this work. My number to feel good about accepting is [X].
> Is there any path to get there?"

## Tone
Collaborative, not adversarial. The goal is a yes both parties feel good about.
